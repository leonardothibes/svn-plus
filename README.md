SVN TAG
=======

Utilitário para gerenciamento de tags do SVN.

Forma de uso
------------

Todos os comandos devem ser executados na raiz do projeto que se quer controlar.

##### Listando as tags

```
svn-tag list
```

##### Criando uma tag

```
svn-tag [nome-da-tag]
```

##### Deletando uma tag

```
svn-tag del [nome-da-tag]
```

Observações Gerais
------------------

 * O script herda a autenticação do ambiente.
